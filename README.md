# YY_BASE_KOA_JS

##这里是 yyue 写的一部基础框架

#安装
npm i

#运行基础程序
npm start

#功能介绍
1、实现了文件上传接口，
2、封装了验证码
3、按照 java 模式对接口进行了文件夹分类
4、提供了密码加密和 token 生成，校验模式
