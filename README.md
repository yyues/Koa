# YY_BASE_KOA_JS

##这里是 yyue 写的一部基础框架

#安装
- npm i

#运行基础程序
- npm start

#功能介绍

- 实现了文件上传接口，
- 封装了验证码
- 按照 java 模式对接口进行了文件夹分类
- 提供了密码加密和 token 生成，校验模式
- 实现登录拦截功能，不登陆就没法进行其他请求
- 短语验证可以实现，基于阿里云的
- 对于返回数据继续了基本的封装，后续可以改进
